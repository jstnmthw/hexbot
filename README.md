# n0xb0t

Obnoxiously intelligent, AI-powered IRC bot.

> **Under construction** — this project is in early development.

## Quick start

```bash
pnpm install
cp config/bot.example.json config/bot.json
# Edit config/bot.json with your server, nick, and owner hostmask
pnpm start
```
