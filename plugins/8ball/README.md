# 8ball

Magic 8-Ball plugin. Ask a yes/no question, get a random answer.

## Usage

```
!8ball Will this work?
```

Responds with one of 20 classic Magic 8-Ball answers (10 affirmative, 5 non-committal, 5 negative),
prefixed with the 🎱 emoji (e.g., `🎱 It is certain.`).

If no question is provided, the bot replies with a usage hint.

## Config

No configuration needed.
